# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - Phase 1 Stable Release
### Added
- Complete Phase 1 architecture setup.
- Initialised Pyproject configuration and virtual environment.
- Added comprehensive Utils module (logger, helpers, formatters, validators, exceptions).
- Added Config module (Pydantic Settings).
- Database Engine (aiosqlite) with automated table initialization.
- Fully defined ORM schema (Users, Conversations, Messages, Tasks, Memories, Agent Logs).
- Initial test coverage spanning Config and Database Repositories.
